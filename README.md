yo.go
====

[Yo](http://www.justyo.co/) to your twitter friend from CLI.

## Usage

Send yo !

```bash
$ yo deeeet
```

## Install

If you use OSX, you can install through brew:

```bash
$ brew tap tcnksm/yo
$ brew install yo
```

If you are another platform install from [releae page](https://github.com/tcnksm/yo/releases)
